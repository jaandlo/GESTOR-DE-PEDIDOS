# Gestor de Pedidos - Frontend

Este proyecto es una interfaz estática para un gestor de pedidos multi-sede.

## Archivos principales

- `index.html` — Estructura de la aplicación
- `styles.css` — Estilos visuales y diseño
- `app.js` — Interacciones y navegación entre pantallas

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Ingresa un usuario y selecciona un rol.
3. Navega entre las vistas de dashboard, creación de pedido, bodega, detalle y historial.

## Notas

- Es una demo de frontend basada en la idea de la aplicación de pedidos.
- Todos los datos son de ejemplo y se generan localmente en el navegador.
